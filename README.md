# Aplicación de cifrado de archivos
Frontend de una aplicación web para el cifrado de archivos de texto y envío de estos entre usuarios

## Ejecución
    npm install

    npm start


## Configuración
Hay que crear un archivo .env en la raíz con la siguiente estructura:

    DB_HOST = 
    DB_USER = 
    DB_PASSWORD = 
    DB_DATABASE = 
    PORT =
